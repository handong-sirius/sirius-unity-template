# 🎮 SIRIUS Unity Project Template

시리우스 학회원들이 **Unity 프로젝트를 시작할 때** 사용하는 **공용 템플릿**입니다.  
프로젝트 시작 전, 이 템플릿을 복사해서 사용하세요.  
(깃허브 **"Use this template"** 기능 활용 권장)

## ✅ 목적
- 프로젝트 초기 세팅 시간을 줄이고, **통일된 폴더 구조**를 유지
- **공용 유틸/코드**를 쉽게 재사용
- 학회원 간 협업 시 **충돌 및 관리 편의성** 향상
